🚀 DC Paczkomaty – Online Payment
Free OpenCart 4 extension that integrates the InPost parcel locker map directly into checkout for online payments.

✨ Features
• 🗺️ Interactive InPost map with parcel locker selection  
• 💳 Works with online payments (PayU, Przelewy24, etc.)  
• 🧩 Fully compatible with Ship2Pay modules  
• ⚙️ Uses VQMod – safe code injection, no core changes  
• 💬 Locker info is automatically saved in the order comment  

📦 Installation
1️⃣ Unzip the file dc_paczkomaty_po.zip  
2️⃣ Upload the included “vqmod” folder to your store root directory  
3️⃣ Install dc_paczkomaty_po.ocmod.zip via Extensions → Installer  
4️⃣ Go to Extensions → Modifications and click Refresh  
5️⃣ Enable the shipping method “Paczkomaty (Online Payment)”  

💡 Requirements
• OpenCart 4.x  
• VQMod installed
